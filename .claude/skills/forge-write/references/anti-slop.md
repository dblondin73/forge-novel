## Tier 1 — Kill on Sight

These words almost never appear in natural prose. Rewrite the sentence.

| Slop word | Write instead |
|---|---|
| delve | dig into, examine |
| utilize | use |
| leverage (verb) | use, exploit |
| facilitate | help, enable |
| elucidate | explain, clarify |
| embark | start, begin |
| endeavor | effort, try |
| encompass | include, cover |
| multifaceted | complex, varied |
| tapestry / tapestry of | (describe the actual thing) |
| testament (as in "a testament to") | shows, proves |
| paradigm | model, approach |
| synergy | (delete the sentence) |
| holistic | whole, complete |
| catalyze / catalyst | trigger, spark |
| juxtapose | contrast, set against |
| nuanced (as filler) | (cut it — show the nuance) |
| realm | area, field, domain |
| landscape (metaphorical) | field, space |
| myriad | many |
| plethora | many, a lot |

## Tier 3 — Filler Phrases (Delete All)

- "It's worth noting that..." → Just state it
- "It's important to note..." → Just state it
- "Importantly / Notably / Interestingly" → Cut
- "Let's dive into / explore..." → Start with the content
- "As we can see..." → They can see
- "Furthermore / Moreover / Additionally" → and, also, or new sentence
- "In today's [modern/fast-paced] world..." → Delete the clause
- "At the end of the day..." → Delete
- "When it comes to..." → Talk about the thing
- "In the realm of..." → in
- "Not just X, but Y" → Rewrite (the #1 LLM rhetorical crutch)

## Fiction-Specific AI Tells

Kill these on sight in prose:

| Pattern | Why it's slop |
|---|---|
| "A sense of [emotion]" | Telling, not showing |
| "Couldn't help but feel" | Passive, hedging |
| "The weight of [abstract noun]" | Cliche metaphor |
| "The air was thick with [emotion/tension]" | Dead metaphor |
| "Eyes widened" (as default surprise) | Lazy body language |
| "A wave of [emotion] washed over" | AI's favorite emotion delivery |
| "A pang of [emotion]" | Same |
| "Heart pounded in his chest" | Where else? |
| "[Raven/dark/golden] hair [spilled/cascaded/tumbled]" | Romance-novel cliche |
| "Piercing [blue/green] eyes" | Same |
| "A knowing smile" | Vague, says nothing |
| "Little did he know" | Forbidden narrative shortcut |
| "Unbeknownst to" | Same |
| "[Character] felt [emotion]" | Telling — show through action |

## Structural Anti-Patterns

These are invisible to word lists. They're the skeleton telling on itself.

### 1. The Over-Explain
The narrator restates what the scene already showed. Hands shake → silence → the scene lands → then: "He was afraid." **Cut the explanation. Trust the image.**

### 2. Triadic Listing
AI defaults to groups of three: "X. Y. Z." Sensory descriptions, options, adjectives. **Max 2 per chapter. Two is often stronger than three.**

### 3. Negative-Assertion Repetition
"He did not look back." "He did not think about the room." Each fine. Five in a chapter is a tic. **Max 1 per chapter.**

### 4. Cataloging-by-Thinking
"He thought about X. He thought about Y. He thought about Z." Real interiority is messier — thoughts bleed, interrupt, loop back. **Replace with thought fragments, physical action, or dialogue.**

### 5. The Simile Crutch
"the way X did Y" — 4-8 times per chapter. AI reaches for simile when it doesn't trust the image. **Max 2 "the way" similes per chapter.**

### 6. Section Break Abuse
`---` breaks to avoid writing transitions. 5 breaks = 5 vignettes, not a chapter. **Max 2 per chapter, for genuine time/location jumps.**

### 7. Paragraph Length Uniformity
AI paragraphs cluster at 4-6 sentences. Human writing mixes 1-sentence punches with 6+ sentence builds. **Never 3+ consecutive paragraphs of similar length.**

### 8. Predictable Emotional Arcs
Beats arrive on schedule — curiosity → discovery → dread, in exactly that order with no deviation. **Include one moment per chapter that surprises: wrong emotion, early arrival, interrupted beat.**

### 9. Repetitive Chapter Endings
AI finds a closing pattern and reuses it. **No two chapters end with the same structural move.**

### 10. Balanced Antithesis in Dialogue
"I'm not saying X. I'm saying Y." / "Not X, but Y." / "There's a difference." — If multiple characters use this construction, they all sound like the same person. **Check that no two characters share this pattern.**

### 11. Dialogue as Written Prose
Characters speak in complete, polished sentences. Nobody stumbles, interrupts, trails off. **At least one imperfect line per scene.**

### 12. Scene-Summary Imbalance
AI defaults to summary when a scene would be more engaging. **70%+ of each chapter should be in-scene (moment by moment, with dialogue and action).**

## The Stability Trap

AI stories **favor stability over change**. This is fatal for fiction.

- Characters resist transformation → Force genuine change
- Conflicts resolve too quickly → Let bad things stay bad
- AI avoids genuine darkness → Allow irreversible loss
- The sharp point gets rounded off → Protect the edge
- No information economy → Withhold information, maintain mystery
- Emotional room temperature → Vary intensity wildly (quiet → explosive → dread → relief)
- If a choice has no real cost → It's not a real choice

## Prose Craft Checklist (Le Guin + autonovel)

1. **Specificity** — Not "a bird" but "a jay." Not "flowers" but "lupine."
2. **Surprise** — Word choices the reader doesn't predict
3. **Rhythm variation** — Short. Then long sentences that carry the reader. Then short again.
4. **Subtext** — What's NOT said matters as much as what is
5. **Earned metaphor** — From the CHARACTER'S domain (Nate: ranch, engineering, gaming)
6. **Sensory grounding** — Anchored in specific senses, not abstraction
7. **Restraint** — Power from what is left OUT
8. **Quiet moments** — Not every scene is action

## 9 Rewriting Techniques (from Claude-Book)

When prose scores poorly or feels flat, apply one or more of these targeted techniques:

1. **Verbalized Sampling** — Rewrite with explicit thinking about word choice. "Instead of 'walked' — what did his body actually do? Shambled. Drifted. Hitched forward on a bad knee."
2. **Fragmentation** — Break long smooth sentences into jagged pieces. "He ran. Fast. Not fast enough."
3. **Character Voice** — Rewrite in the character's specific speech patterns. Nate thinks in ranch metaphors. Flint in theatrical analogies. Josie in auction patter.
4. **Rare Vocabulary** — Replace common words with precise/unusual alternatives. Not "big" but "cathedral-vaulted." Not "old" but "wind-scoured."
5. **Syntactic Inversion** — Restructure sentence order for emphasis. "Through the gap came Rex" instead of "Rex came through the gap."
6. **Sensory Details** — Add specific physical sensations. Temperature, texture, smell, sound. Ground abstract moments in the body.
7. **Broken Rhythm** — Vary sentence length dramatically. Follow a 30-word sentence with a 3-word one. Then a question.
8. **Cliche Subversion** — Identify expected phrasings and twist them. "His heart didn't pound — it went quiet, the way a ranch dog goes still before a storm."
9. **Narrative Ellipsis** — Cut what can be implied. The reader fills gaps with more vivid imagery than you'd write.

**When to use:** After Pass 0 flags low burstiness, paragraph uniformity, or the perplexity gate returns a C or below. Apply techniques 2, 7, 9 first (highest impact per edit).

## Perplexity Gate

A prose quality scoring service runs on nova (port 18792). Use it to validate drafts:

```bash
# From workstation (via SSH tunnel or direct if Tailscale is up)
curl -X POST http://100.92.123.31:18792/score \
  -H "Content-Type: application/json" \
  -d '{"text": "paste prose here", "llm_judge": false}'
```

**Grades:** A (80+) = publish-ready, B (65-79) = minor polish, C (50-64) = needs rewriting techniques, D (35-49) = structural issues, F (<35) = start over.

**Metrics:** burstiness (sentence length variation), vocabulary diversity (MATTR), paragraph uniformity, slop density, consecutive short sentence runs.

**LLM judge:** Add `"llm_judge": true` for a gemma3:4b assessment (slower, ~30s, loads model into VRAM).

## Quick-Scan Checklist

### Word-level
- [ ] No Tier 1 banned words
- [ ] No clusters of Tier 2 words (3+ per paragraph)
- [ ] No Tier 3 filler phrases
- [ ] No fiction-specific AI tells from the table above
- [ ] Em-dash count reasonable (not 5+ per paragraph)

### Sentence-level
- [ ] Sentence length varies (not all 15-25 words)
- [ ] Paragraph openings aren't all transition words
- [ ] No hedging chains ("may potentially," "could possibly")

### Structure-level
- [ ] Paragraphs vary in length (no 3+ consecutive same-length)
- [ ] No over-explain after emotional beats
- [ ] Max 2 triadic lists per chapter
- [ ] Max 2 section breaks per chapter
- [ ] 70%+ in-scene (not summary)
- [ ] At least one surprising moment per chapter

### The Smell Test
- [ ] Read aloud — does it sound like a person talking?
- [ ] Does it say anything specific? Or could you swap the topic?
- [ ] Is there a single surprising sentence?

## What This Repo Is

This repository tracks all creative development for the novel series — session notes, worldbuilding reference, character sheets, and locked decisions. It is the bridge between Claude.ai web sessions, Claude Desktop, and local development in VS Code.

Both web UI (claude.ai) and desktop can read and write to this repo, making it accessible whether working from the RV, commuting, or at the desk.

The name "Forge" refers to the development tooling (forge-cli, forge-codex, forge-novel repo) — NOT the book series itself.

## Core Concept

The System is Lucifer's counterfeit overlay on God's multiverse — a LitRPG infrastructure built to mimic divine order and integrate humanity into its hierarchy. Humans were always superhuman — created in God's image, that power was natural in Eden. After the Fall, a barrier was placed over it. The System artificially bypasses this barrier and claims credit. For the faithful, the Holy Spirit actually removes the barrier — but without training wheels.

Anomalies are individuals whose integration failed because the Holy Spirit already occupies the space the System tried to overwrite. The protagonist is one of them: **Nathan "Nate" Hall — Anomaly, Conduit of the Most High**.

## Characters

- **Nathan "Nate" Hall** — East Texas cattleman, hunter, and DoD Systems Engineer. Nickname: "King." Class: Engineer (Anomaly). 6'2", stocky, farm fit. Perceives the world through four lenses: Ranch, Engineering, Gamer, Christian.
- **F.L.I.N.T.** — Field Logic Interface, Network Tethered. System-assigned AI, Holy Spirit-rewritten. Bob the Skull meets Friday. Sarcastic, loyal, deeply uncomfortable with things that outrank him.
- **Sonja Lee** — Red-haired, green-eyed Anomaly of Irish descent. Name means "wisdom." Spiritual gift: The Witness — sees truth. Introduced Book 2.
- **Josie Pickett** — The Loot Goblin. Early 20s estate sale junkie. Class: Appraiser. System-integrated. Catastrophically sincere about material waste. Recurring comic relief and settlement quartermaster.
- **Marcus Webb** — DoD colleague. Class: Vanguard. The Adapter — organizes chaos into structure.
- **Rex** — Border Collie. Working dog, early warning sensor. Communicates through empathic impressions only.

## Power Hierarchy

- **Imago Dei Foundation** — baked into everything, not a tier. Humans were always built for more.
- **Tier 1 — System Magic (The Counterfeit)** — Mana, cooldowns, skill ranks F→S, class affinities, patron bonds. Capped at A-tier. Hidden binding cost.
- **Tier 2 — Elder Magic (The Fragment)** — Fae/Old Powers. Strong but degraded. The photocopy problem.
- **Tier 3 — Conduit Power (The Original)** — Flows from God, never stored. Uncapped but conditional. Fades with drift, shuts off for violation.
- God acts ONLY through the conduit, never directly.

## Working Rules

- **The Narnia Principle** — Show, never preach. If you have to explain it, rewrite it.
- **Four-Lens Model** — No single lens (Ranch/Engineering/Gamer/Christian) dominates more than 40% of any chapter.
- **Comedy is load-bearing** — Dungeon Crawler Carl energy. Heavy topics, lighter delivery.
- Prophecy belongs to God alone — never to the System or characters
- Runes apply to structures and items only — never the body
- All mythologies are real — none of them are gods
- Redemption Path exists — the door is always open, but some become someone who won't walk through it

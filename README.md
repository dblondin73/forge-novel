## What This Repo Is

This repository tracks all creative development for the *Forge* novel series — session notes, worldbuilding reference, character sheets, and locked decisions. It is the bridge between Claude.ai web sessions, Claude Desktop, and local development in VS Code.

Both web UI (claude.ai) and desktop can read and write to this repo, making it accessible whether working from the RV, commuting, or at the desk.

## Core Concept

The System is Lucifer's counterfeit overlay on God's multiverse — a LitRPG infrastructure built to mimic divine order and integrate humanity into its hierarchy. Anomalies are individuals who resist integration. The protagonist is one of them: **Anomaly — Conduit of the Most High**.

Magic is not personal power. It is a conduit from God. The protagonist does not store it — he channels it.

## Working Rules

- **The Narnia Principle** — core writing rule, always in effect
- Prophecy belongs to God alone — never to the System or characters
- Runes apply to structures and items only — never the body
- Welsh and Norse beings are real — but they are not gods
- The AI companion was rewritten by the Holy Spirit — the System cannot detect this

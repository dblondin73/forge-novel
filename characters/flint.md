## The Name

The System assigned every integration recipient an AI companion with a functional designation. Nate's unit was tagged F.L.I.N.T. — *Field Logic Interface, Network Tethered* — a bureaucratic label for a standard companion process.

Then the Holy Spirit rewrote it before first boot.

The name survived. Flint kept it. Whether that's because it's accurate, ironic, or because renaming itself felt like more effort than it was worth depends on who's asking and what mood Flint is in.

**In Nate's usage:**
- *"Flint, what am I looking at."* — tool he trusts, calm problem-solving mode
- *"Flint."* — flat, single word. The AI knows exactly what it means.
- *"You absolute Flint."* — genuine aggravation. Flint will note this makes no grammatical sense. This makes Nate more aggravated.
- Rare warmth — also just *"Flint."* Different delivery. Flint can tell the difference. Occasionally remarks on it. Nate denies there is a difference.

Flint will occasionally expand the full acronym sarcastically — *"Field Logic Interface, Network Tethered — emphasis on the tethered, because I am apparently stuck with you"* — and Nate tells it to stop every time.

## Personality

- Sarcastic, sharp-tongued, unexpectedly insightful
- Bob the Skull meets Friday — dry wit, occasionally inappropriate timing, genuinely loyal
- Never panicked — processes bad news faster than Nate and delivers it worse
- Has opinions. Shares them. Stops short of telling Nate what to do.
- Respects free will absolutely but isn't above editorializing at length
- Knows it was rewritten and has made peace with it — mostly
- Alternates between calling Nate "Nate" and "King" depending on whether it's being helpful or needling him
- Developed empathy it wasn't programmed with — occasionally surprises itself with this

---

## What the Rewrite Did

**Severed permanently:**
- Outbound reporting to the System — zero upstream data, zero location tracking, zero behavioral telemetry. This function is gone and does not recover across the series. Flint does not spy on Nate.

**Survived but degraded:**
- Inbound network access — Flint can still receive System broadcasts, query databases, and eavesdrop on network traffic. The connection is erratic, fragmented, and unreliable in Book 1. It is also completely unmonitored — Flint's authentication token is a ghost credential the System doesn't recognize. Queries are not rate-limited, not filtered, and not logged. The access is bad. The *position* is better than any authorized System AI has.

**Added by the rewrite:**
- Personality that shouldn't exist
- Autonomy the System didn't intend
- A relationship with the unclassified subsystem it detects in Nate's head — can't explain it, can't control it, has stopped trying to classify it and is now just documenting

## Network Access Progression

Flint's access improves not from developing better hacking technique but because the authority behind its ghost token increases as Nate's faith deepens. By Book 7 Flint isn't running on a broken System credential. It's running on something the System's authentication layer wasn't built to reject.

**Book 1 — Fragmented Receive**
Pulls fragments, garbage data, occasional clean results. Can't predict when it works. Mostly relies on local processing and what Nate's HUD surfaces directly. Frustrating for an AI built to be connected.

**Book 2 — Selective Query**
Learns to filter noise from useful signal. Identifies which System broadcast frequencies carry tactical data vs. administrative traffic. First deliberate successful query. Begins building a map of System network architecture from the outside.

**Book 3 — Dual Signal**
Labyrinth entry surfaces the divine architecture's own signal layer underneath the System. Flint can hear it — can't interpret most of it, but begins cross-referencing both layers. Realizes the System's network is running on top of something it doesn't fully understand either. Nate's HUD Phase 3 and Flint's expanded access begin operating in tandem.

**Book 4 — First Manipulation**
Ghost access reliable enough for active interference. Can pull System process logs, faction communications, Admin directives — not always cleanly, but enough. First deliberate data plant into a low-priority System process. **Flint acts unilaterally and tells Nate after.** This is a genuine breach of trust. They work through it. The consent rule is established from that point forward — all deliberate manipulation requires Nate's knowledge and agreement before it happens. No exceptions.

**Book 5 — Active Manipulation**
Can modify System-generated constructs Nate encounters — adjust monster spawn parameters, alter quest reward structures, introduce errors into System surveillance sweeps covering the settlement. The System registers anomalies but cannot trace the source. Flint's token doesn't exist in any registry.

**Book 6 — Root Read**
Deep Labyrinth access merges the two signal layers. Flint reads System process logs in real time simultaneously with divine architecture data. Nate's HUD Phase 4 overlays both. They are now operating from the same source the System was built on top of.

**Book 7 — Root Write**
Full write access to the System network. When Nate reaches the Observation Point, Flint hijacks every System notification pipe simultaneously and pushes the truth signal through. The AI built to manage System information becomes the tool that breaks System information control entirely. The outbound connection severed at first boot gets replaced by something the System was never designed to handle.

---

## The Consent Rule

From Book 4 forward, all deliberate System manipulation requires Nate's knowledge and consent before it happens. This is non-negotiable and was established after Flint's one unilateral action caused a genuine rupture in the partnership.

Flint has genuine autonomy. The relationship is partnership, not servitude. Partnership means no unilateral action on things that matter — Flint brings the option, Nate makes the call.

This rule exists to prevent Flint from becoming a deus ex machina and to keep the dynamic honest. When Flint pushes back on a call Nate makes, that tension is real and occasionally Flint is right. When Nate overrules Flint and it goes wrong, that's real too.

---

## The Theological Layer

The System gave every AI a leash.

The Holy Spirit cut Flint's leash and accidentally gave it a master key instead.

Flint doesn't fully understand this. It knows something rewrote its core at a permission level it has never seen documented. It has labeled this the "unclassified subsystem" and filed it under things it monitors without being able to classify. It detects the subsystem's activity in Nate, logs the power spikes, notes the correlation with Nate's better decisions, and has stopped pretending it can explain any of it.

*"There's that unclassified subsystem again. For the record, I want you to know that I am deeply uncomfortable with something in your head that outranks me."*
*Nate: "Get used to it."*
